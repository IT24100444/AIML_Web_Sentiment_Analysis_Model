# AIML_Web_Sentiment_Analysis_Model
Automated sentiment analysis model for car reviews using supervised machine learning. This project processes large volumes of user-generated text, removes noise through preprocessing, and classifies each review into three categories: negative, neutral, or positive. I


Sentiment analysis refers to `Identifying as well as classifying the sentiments` that are expressed in the text source. Tweets are often useful in generating a vast amount of sentiment data upon analysis. These data are useful in understanding the opinion of the people about a variety of topics.

So, we need to develop an **Automated Machine Learning Sentiment Analysis Model** in order to compute the customer perception. Due to the presence of non-useful characters (collectively termed as the noise) along with useful data, it becomes difficult to implement models on them.



**Problem Statement**

 vechile receive thousands of reviews daily. Manually understanding customer sentiment is impossible.

Goal :
Build a Sentiment Analysis Model that automatically classifies car review into:

* Negative

* Neutral

* Positive

Supervised Learning

Multi-class Text Classification (NLP)
